# 免开发广告投放执行单（微信流量主助手）

## 1. 页面投放策略
- 禁投页面：`create`、`lobby`、`settings`、`schedule`、`match`
- 可投页面：`home`、`analytics`
- 备选灰度页面：`ranking`（仅在收益不足时开启）

## 2. 点位与节奏
- `home`：1 个原生模板位，放在“进行中/草稿”与“历史”之间
- `analytics`：1 个原生模板位，放在“总览”与“球员数据”之间
- 开屏：每日首次 1 次

## 3. 频控规则（后台配置）
- 单用户单日开屏上限：1
- 单会话总曝光上限：2（`home=1` + `analytics=1`）
- 同页曝光冷却：30 分钟
- 用户两次快速关闭/跳出后：次日降频

## 4. 灰度与放量
- 第 1 阶段：10% 灰度 3 天
- 第 2 阶段：观察指标稳定后全量
- 每周一次调优：仅后台参数，不改代码广告组件

## 5. 验收指标
- 开赛率下降 <= 2%
- 录分完成率下降 <= 1%
- D1 留存下降 <= 1.5pp
- 广告关闭率异常升高时，优先降频

## 6. 回滚顺序
1. 关闭开屏
2. 降低/关闭 `home` 位
3. 仅保留 `analytics` 位

## 7. 当前代码侧已配合完成
- `home` / `analytics` 已预留自然广告断点，不打断主流程
- 本地守卫已就绪：同页 30 分钟冷却、会话上限 2、每日开屏标记
- 未接入激励视频/插屏/支付相关逻辑，符合“免开发广告”策略
